## 🏃💨 Getting started

1. Clone this repo with `git clone https://github.com/calebesg/my-repos.git`
2. Move yourself to the my-repos directory: `cd my-repos`
3. Run the `npm install` add all dependencies
4. Run `npm run dev` start app
